My Work

Social Media Clone (Frontend)

Frontend Interface mimicking Instagram's design, using React, React Router for navigation, along with Chakra UI and FontAwesome for responsive design and icons.

Social Media Clone (Backend)

I developed the backend using Node.js, Prisma for database management, Multer for handling file uploads, and Cloudinary for image storage. Authentication is managed with Passport and bcryptjs for secure password hashing.

Blog (Frontend)

I built the blog's frontend using React and React Router for smooth navigation. Users can browse posts, create accounts, like, and comment on posts. Authors have the ability to add, edit, and delete their own posts, all within a clean and responsive UI.

Blog (Backend)

The backend is developed with Node.js and PostgreSQL for database management. It handles user authentication, post management, and interactions like comments and likes, ensuring secure and efficient data flow across the platform.

Weather App

Before learning React, I built a weather app that makes API calls to fetch real-time weather data. The app displays current weather conditions based on user input, providing a simple yet functional interface for users to check forecasts.
